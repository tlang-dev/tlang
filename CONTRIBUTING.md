# Contributing to TLang

Thank you for your interest in contributing! The full contributing guide lives at
[`docs/contributing/CONTRIBUTING.md`](docs/contributing/CONTRIBUTING.md).

## Quick start

```bash
git clone https://github.com/joel-f/tlang.git
cd tlang
cargo build
cargo test
```
